一頁式速記表

  • (1) 三大變位先記住
    -ar（第一變位，如 hablar 說話）/ -er（如 comer 吃、beber 喝）/ -ir（如 vivir 居住／生活）
  • (2) 第一變位 -ar（現在式）
    字尾：yo -o / tú -as / él-ella -a / nosotros -amos / vosotros -áis / ellos-ellas -an；hablar 說話 → hablo, hablas, habla…
  • (3) 第二變位 -er（現在式）
    字尾：yo -o / tú -es…；comer 吃 → como, comes, come…
  • (4) 主詞可省略（超常考）
    cocinar 煮飯／烹調：Yo cocino.（可） / Cocino.（更常見）
  • (5) gustar 核心句型（最重要）
    gustar 喜歡（句型：使某人喜歡）；公式：me/te/le… + gusta(n) + 主詞；例：Me gusta el chocolate.、Te gustan los dulces.
  • (6) encantar 用法同 gustar
    encantar 非常喜歡／愛極了：Me encanta... / Me encantan...

一頁式速記表

  • (1) 定冠詞
    el / la / los / las（這個／那個／這些／那些）
  • (2) 名詞陰陽性常見尾碼
    -a 常陰、-o 常陽（有例外）
  • (3) 常見例外
    la mano 手、el agua 水（陰性名詞）
  • (4) 複數規則
    母音結尾 + s；子音結尾 + es
  • (5) -ir（vivir）現在式
    vivir 居住／生活：vivo, vives, vive, vivimos, vivís, viven
  • (6) también / tampoco
    también 也；tampoco 也不（呼應否定句）
  • 常見錯誤快修正
    la problema ✗ → el problema 問題 ✓；hablar 說：No hablo japonés, yo también ✗ → ... yo tampoco ✓

  • (1) muy 用法
    muy 很／非常 + 形容詞/副詞（不變形）
  • (2) mucho 用法

一頁式速記表

  • (1) 反身動詞結構
    動詞原形以 -se 結尾（如 llamarse 叫做、levantarse 起床）；使用時搭配反身代名詞：me, te, se, nos, os, se
  • (2) llamarse 變位
    me llamo, te llamas, se llama, nos llamamos, os llamáis, se llaman
  • (3) levantarse 變位
    me levanto, te levantas, se levanta, nos levantamos, os levantáis, se levantan
  • (4) 日常作息常用反身動詞
    despertarse 醒來（e→ie）、ducharse 淋浴、acostarse 就寢（o→ue）、vestirse 穿衣（e→i）
  • (5) 時間副詞
    por la mañana 早上；por la tarde 下午；por la noche 晚上；siempre／a veces／nunca 總是／有時／從不
  • (6) 反身 vs 非反身
    lavar la ropa（洗衣服）vs lavarse（洗澡/洗手）；動作作用在自身 → 加 -se

Clase 2-2 — ESPANOL SI DVD 02 clase 06

影片

起始時間 30:29（1829s）。

在 YouTube 開啟

一頁式速記表

  • (1) ser 用於永久／本質
    國籍、職業、材質、身份：Soy médico.（我是醫生）；Es de madera.（是木頭做的）；Son las tres.（現在三點）
  • (2) estar 用於狀態／位置
    位置、當下狀態、情緒：Estoy en casa.（在家）；Estás cansado.（你累了）；El café está frío.（咖啡冷了）
  • (3) estar 變位
    estoy, estás, está, estamos, estáis, están（注意重音）
  • (4) 形容詞與 ser/estar 差異
    Es aburrido.（他這人無聊）vs Está aburrido.（他現在很無聊）；Es malo.（品質差）vs Está malo.（他病了）
  • (5) hay 表示「有」（存在）
    hay 不變形：Hay un banco.（有一間銀行）；Hay muchos turistas.（有很多遊客）；和 está/están（已知物體位置）不同
  • (6) 記憶口訣

一頁式速記表

  • (1) 現在進行式公式
    estar（變位）+ gerundio：表示「現在正在做」；Estoy comiendo.（我正在吃）；Estamos hablando.（我們正在說話）
  • (2) Gerundio 構成：-ar 動詞
    去掉 -ar → 加 -ando：hablar → hablando、trabajar → trabajando、estudiar → estudiando
  • (3) Gerundio 構成：-er/-ir 動詞
    去掉 -er/-ir → 加 -iendo：comer → comiendo、beber → bebiendo、vivir → viviendo
  • (4) 不規則 gerundio（常用）
    ir → yendo；leer → leyendo（母音間 i→y）；decir → diciendo（e→i）；pedir → pidiendo（e→i）；dormir → durmiendo（o→u）
  • (5) 直接受詞代名詞
    lo / la / los / las（取代已知受詞）：¿Ves el libro? — Sí, lo veo.（你看到書嗎？→是，我看到它）
  • (6) 代名詞可接在 gerundio 後
    Estoy leyéndolo.（我正在讀它）= Lo estoy leyendo.（兩種寫法皆正確）

一頁式速記表

  • (1) 近未來：ir a + infinitivo
    公式：ir（變位）+ a + 動詞原形；Voy a comer.（我要去吃／我打算吃）；Vamos a estudiar.（我們要學習）
  • (2) ir 近未來變位對照
    voy a / vas a / va a / vamos a / vais a / van a + 動詞原形
  • (3) 時間副詞
    mañana 明天；esta tarde/noche 今天下午/晚上；el próximo lunes 下週一；la semana que viene 下週；el año que viene 明年
  • (4) 比較級
    más + 形容詞/副詞 + que（比⋯更）；menos + 形容詞 + que（比⋯少）；tan + 形容詞 + como（和⋯一樣）
  • (5) 比較級不規則
    bueno → mejor（更好）；malo → peor（更差）；grande → mayor（更大/更年長）；pequeño → menor（更小/更年幼）
  • (6) 最高級
    el/la más + 形容詞：Es el más inteligente.（他是最聰明的）；Es la mejor.（她是最好的）
